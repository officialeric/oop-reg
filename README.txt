## About

This project is a simple registration system implemented in PHP using Object-Oriented Programming (OOP) principles. It provides a basic framework for user registration, including form validation, user authentication, and database storage.

## Features

- User registration with validation
- Secure password hashing
- User login and authentication
- Database integration for storing user data
- Customizable and extensible for your own use cases

## Requirements

To run this registration system locally, you'll need:

- PHP (version latest)
- MySQL or another database system
- A web server (e.g., Apache, Nginx)
- Composer (for managing PHP dependencies)

## Installation

1. Clone the repository to your local machine:

   ```bash
   git clone https://github.com/officialeric/oop-reg.git

2. Change to the project directory:

   ```bash
  cd registration-system-php

3. Install PHP dependencies using Composer:

   ```bash
   composer install

4. Configure your database settings by copying the config/config.example.php file to config/config.php and updating the values:

   ```bash
   cp config/config.example.php config/config.php

5. Import the database schema located in the database directory into your MySQL database.

6 .Start your web server and open the project in your browser.

## Usage

-Register as a new user by filling out the registration form.
-Log in with your registered credentials.
-Customize and extend the system to fit your specific requirements.
